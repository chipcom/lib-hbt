# FunLib

Библиотека для Harbour

Классы:

**TFileText** - объект выводящий текстовую информацию в файл

Функции:

**AlertX(** *param, description* **)** - вывод информационного окна для *param* с комментарием *description*.

**Split(** *cStr, sep* **)** - преобразовать строку *cStr* в массив с учетом разделителя *sep* (по умолчанию пробел).

**splitStrToArray(** *cStr, n, sep* **)** - преобразовать строку *cStr* в массив с учетом разделителя *sep* (по умолчанию пробел), причем длина строк в полуившемся массиве не должна превышать *n*.

**win_AlertX(** *param* **)** - вывод окна Windows со значением *param*.

`win_AlertX( param )`
